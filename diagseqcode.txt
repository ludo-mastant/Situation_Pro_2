sequenceDiagram
    autonumber
    participant Admin as Administrateur
    participant App as App Flutter
    participant API as API Laravel
    participant DB as Base de données MySQL


    Admin->>App: Ouvre le formulaire "Ajouter puzzle"
    App->>Admin: Affiche les champs (nom, description, image, prix, catégorie, stock)
    
    Admin->>App: Remplit le formulaire et clique "Créer"
    App->>App: Valide les champs (non vide, prix numérique, etc.)
    
    App->>API: POST /puzzles {nom, description, image, prix, categorie_id, stock}
    API->>DB: INSERT INTO Puzzles (...)
    DB-->>API: Retourne succès / id du puzzle
    API-->>App: Réponse JSON {success, puzzle}
    
    App-->>Admin: Affiche confirmation "Puzzle ajouté avec succès"
